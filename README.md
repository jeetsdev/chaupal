# Chaupal
  A Social Media App 
</div>

## **How to install and run locally ?**

```
$ git clone https://github.com/jeetsdev/chaupal
$ cd chaupal
$ npm install
$ npm start
```
## **Features -**

- Landing Page
- Feed Posts
- Explore All Posts
- Sort by Date, Recent and Trending
- Add , Update & Delete Post
- Add & Delete Comment
- User Profile Update
- Follow / Unfollow
- Like and Comment on Post
- Add & Remove from Bookmark
- Authentication:
  - User Signup
  - User Login
  - User Logout
 - Fully Responsive

## **Built with -**

- React JS
- Tailwind CSS
- Redux Toolkit 
- React Router v6
